# Ultrasonic Distance LED

This Arduino project measures distance using an ultrasonic sensor and controls an LED based on the measured distance.

## Components Used
- Arduino board
- Ultrasonic sensor (HC-SR04)
- LED
- Resistors
- Jumper wires
- Breadboard
